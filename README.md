# The Metropolitan Museum of Art — Landing Page

This project is a responsive landing page for The Metropolitan Museum of Art (The Met). It includes museum hours, current exhibitions, an online exploration section, and a contact form.

## Live Preview

[Live Demo](https://eg598.github.io/The-Met/)

## Technologies Used

**Core**
- HTML5 — Semantic markup
- SCSS — Styling with variables, mixins, and modular structure
- JavaScript (Vanilla) — Burger menu interaction

**Build Tools**
- Parcel — Bundler and dev server
- Stylelint — CSS/SCSS linting
- ESLint — JavaScript linting

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/eg598/The-Met.git
cd The-Met
```

2. Install dependencies:

```bash
npm install
```

3. Run the project locally:

```bash
npm start
```

4. Build for production:

```bash
npm run build
```

## Features

- **Responsive Design** — Optimized for mobile, tablet, and desktop screens
- **Burger Menu** — Smooth navigation menu with open/close toggle
- **Museum Hours** — Up-to-date schedule for both Met locations
- **Now on View** — Current exhibitions with images and dates
- **Explore Online** — Cards linking to online collection areas
- **Contact Form** — Name, email, and message fields with basic validation
